02 · Key findings
Other Relevant Info: near-total failure
80.7% error rate (F1 = 0.285). The model misroutes 353 samples to "Not Humanitarian." This catch-all class has no discriminative surface features for zero-shot prompting.
Confidence scores are not calibrated
59% of all errors occur at confidence ≥ 0.90. Mean confidence gap between correct (0.918) and wrong (0.881) predictions is only 0.037 — confidence cannot be used as a quality filter.
Requests/Needs: rare-class collapse
63.6% error rate on just 55 samples. Model defaults to Rescue/Donation (28 errors) due to keyword overlap. Class imbalance amplifies zero-shot failure on minority classes.
Injured/Dead is the strongest class
F1 = 0.893 on 1,362 samples. Death-count and casualty language is unambiguous and well-represented in the model's pretraining data.
Not Humanitarian bleeds into Sympathy
68 Not-Humanitarian tweets predicted as Sympathy/Support. Political commentary about the disaster shares surface-level emotional language with genuine sympathy expressions — the boundary is too subtle for zero-shot.
Missing People confused with Injured/Dead
32 / 125 missing-person tweets predicted as Injured/Dead. A critical operational error — co-occurrence of names and disaster context drives the model toward the larger, more frequent fatality class.
